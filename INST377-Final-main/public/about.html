About This Project

The Mission

Developed for INST 377, this application serves as a Financial Growth Simulator. It empowers users to visualize the real-world impact of a $100 investment over time, moving beyond simple price tracking into meaningful data analysis.

Note on API Limitations: The free tier of the Alpha Vantage API may limit watchlist updates and calculations in real-time(leaving the predicted price at $100), though the underlying logic remains functional.

The Tech Stack

Backend: Node.js & Express.js
Database: Supabase (PostgreSQL)
External API: Alpha Vantage
Visualization: Chart.js

Mathematical Implementation

The core logic utilizes a time-series calculation to determine Return on Investment (ROI):

$V_{current} = 100 \times \left( \frac{P_{current}}{P_{initial}} \right)$
